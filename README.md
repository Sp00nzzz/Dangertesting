# DangerTestingVirus

A React web app with Tailwind CSS featuring an atmospheric green-lit hallway background.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Add your image:
Place your hallway image at `src/assets/hallway.jpg`

3. Run the development server:
```bash
npm run dev
```

4. Build for production:
```bash
npm run build
```

## Tech Stack

- React 18
- Vite
- Tailwind CSS
- PostCSS & Autoprefixer

## Project Structure

```
src/
  ├── assets/        # Images and static assets
  ├── App.jsx        # Main app component
  ├── main.jsx       # React entry point
  └── index.css      # Tailwind imports and global styles
```
